## 🔐 Form Validation Rules

### Full Name
- Minimum 3 characters
- Pre-validation: On blur
- Error message: "Please enter a valid full name"

### Email
- Format: `user@domain.com`
- Regex: `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
- Pre-validation: On blur
- Error message: "Please enter a valid email address"

### Phone Number
- Exactly 10 digits
- No spaces or special characters
- Auto-formatting as user types
- Error message: "Mobile number must be exactly 10 digits"

### Subject
- Minimum 1 character
- Pre-validation: On input
- Error message: "Please provide a subject"

### Service
- Must select from dropdown
- Required field
- Error message: "Please select a service"

### Message
- Minimum 10 characters
- Pre-validation: On input
- Error message: "Message must be at least 10 characters"

### Terms & Conditions
- Must be checked
- Required checkbox
- Error message: "You must agree to the terms and conditions"

## 🚀 Deployment Instructions

### Quick Deploy to GitHub Pages

1. **Create GitHub Account** → [github.com](https://github.com)

2. **Create Repository**
   - Go to New Repository
   - Name: `portfolio` or `yourusername.github.io`
   - Select Public

3. **Push Code**
   ```bash
   cd c:\fsd assign3
   git remote add origin https://github.com/USERNAME/portfolio.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**
   - Settings → Pages
   - Source: main branch, /(root) folder
   - Save

5. **Live URL**
   - `https://yourusername.github.io/portfolio`

**See DEPLOYMENT.md for detailed instructions**
